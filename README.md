# Прогноз риска инсульта

Проект представляет собой веб-приложение, разработанное с использованием Streamlit, которое позволяет медицинским специалистам вводить данные пациента и получать прогноз риска инсульта на основе обученной модели градиентного бустинга. Приложение подключено к репозиторию на GitHub и развернуто в облаке Streamlit Cloud для удобного доступа и использования.
