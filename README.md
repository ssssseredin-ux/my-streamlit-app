Прогноз риска инсульта
Небольшое интерактивное веб-приложение на Streamlit для медицинских работников, позволяющее оценить вероятность инсульта у пациента на основе его демографических и клинико-анамнестических данных.

2. Начало работы
Клонирование репозитория
bash

git clone https://github.com/your-username/stroke-risk-forecast.git
cd stroke-risk-forecast
Установка зависимостей
Рекомендуется создать и активировать виртуальное окружение:

bash

python3 -m venv venv
source venv/bin/activate       # Linux/macOS
venv\Scripts\activate          # Windows
Затем установить пакеты:

bash

pip install -r requirements.txt
3. Использование
Запуск локально
bash

streamlit run app.py
После запуска откройте в браузере адрес, который укажет Streamlit (обычно http://localhost:8501).

Пример ввода
Выберите пол пациента
Укажите возраст (ползунок 18–100)
Отметьте наличие/отсутствие гипертензии и сердечных заболеваний
Задайте семейное положение, тип работы, тип проживания
Укажите средний уровень глюкозы
Выберите статус курения
Нажмите «Оценить риск инсульта»
Приложение выдаст бинарный прогноз (низкий/высокий риск) и вероятность события.

6. Deploy и CI/CD
Streamlit Cloud
Зарегистрируйтесь на https://streamlit.io/cloud
Подключите GitHub-репозиторий
Укажите ветку и файл app.py
Нажмите «Deploy» — приложение будет автоматически перезапускаться при пушах в указанный branch.
GitHub Actions (опционально)
Можно настроить CI-пайплайн для линтинга и тестов. Добавьте в .github/workflows соответствующий YAML-файл.
